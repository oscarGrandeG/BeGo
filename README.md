
# BeGo

## Running On Device

- [React Native Config](https://reactnative.dev/docs/running-on-device) - React Native

## Installation

IOS

```sh
npm i
cd ios
pod install
npx react-native run-ios
```

Android

```sh
npm i
npx react-native run-android
```
